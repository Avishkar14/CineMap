# CineMap Architecture

## Overview

CineMap is a web application that visualizes relationships between movies.

Architecture:

React Frontend
        ↓
Spring Boot REST API
        ↓
TMDB API
        ↓
Recommendation Engine
        ↓
Interactive Graph (Cytoscape.js)

## Future Improvements

- PostgreSQL
- Neo4j
- User Authentication
- Custom Recommendation Algorithms
