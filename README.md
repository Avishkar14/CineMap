# CineMap
CineMap is graph based visualization and interactive movie/series recommendation project.
An interactive movie relationship explorer that visualizes connections between movies using graph-based visualization and custom similarity algorithms.

---

## 🚀 Project Overview

CineMap helps users discover movies in a more intuitive way than traditional recommendation lists.

Instead of showing recommendations as a list, CineMap displays them as an interactive graph where each movie is represented as a node and relationships between movies are represented as edges.

The project is inspired by MALMap but is designed for movies and TV shows with a custom recommendation engine.
